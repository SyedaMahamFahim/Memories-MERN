# Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Memories-MERN.git
```

Go to the project directory run backend by 

```bash
  cd backend-server
```

Install dependencies

```bash
  npm install
```


Start the server

```bash
  nodemon index.js
```
